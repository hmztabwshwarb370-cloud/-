نسخة Golden Star Local Mobile Edition

الفكرة:
- النظام يعمل محلياً على الموبايل أو الكمبيوتر.
- لا يعتمد على Google Sheets أو Apps Script.
- البيانات محفوظة داخل المتصفح نفسه عبر IndexedDB.
- سريع جداً، والحذف والإيرادات والأقساط يتم حسابها محلياً بدون تأخير.

ملفات التشغيل:
- index.html
- style.css
- app.js
- manifest.json
- sw.js
- icon.svg

طريقة النشر الأفضل على GitHub Pages:
1) ارفع كل ملفات هذا المجلد إلى Repository جديد أو نفس المستودع.
2) فعّل GitHub Pages.
3) افتح الرابط من الموبايل مرة واحدة والإنترنت موجود.
4) من المتصفح اضغط Add to Home Screen / إضافة إلى الشاشة الرئيسية.
5) بعد ذلك يعمل النظام كتطبيق، ومعظم الأقسام تعمل بدون إنترنت.

بيانات الدخول:
admin / admin123
finance / finance123

مهم جداً:
- لا تمسح بيانات المتصفح قبل عمل نسخة احتياطية.
- من قسم النسخ والبيانات صدّر نسخة JSON يومياً.
- يمكن تصدير Excel للاطلاع والجداول، لكن النسخة التي تعيد الاستيراد هي JSON.

واتساب:
- إرسال الإيصال أو التذكير عبر واتساب يحتاج إنترنت وقت الإرسال فقط.

ملاحظة:
إذا فتحت index.html كملف مباشر من الهاتف قد تعمل الواجهة، لكن أفضل وأضمن طريقة هي GitHub Pages حتى تعمل PWA والأوفلاين بشكل صحيح.
